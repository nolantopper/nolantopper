## Featured Projects

### [SLAM Estimator Comparison](https://github.com/nolantopper/SLAM-Estimator-Comparison)

Compared state-estimation pipelines for downstream 3D Gaussian Splatting reconstruction. My work focused on running OpenVINS on dataset sequences, using Docker to run a Linux / ROS 2 Humble environment on macOS, running a headless estimator workflow, and matching estimator timestamps to camera poses for trajectory comparison.

**Tools:** ROS 2 Humble, Docker, OpenVINS, Python, Linux, trajectory alignment, state estimation

### [Airfoil Optimization](https://github.com/nolantopper/Airfoil-Optimization)

Built a Python-based airfoil optimization tool using a Hess-Smith panel method and NACA airfoil parameterization. The project uses multi-start Nelder-Mead optimization to explore aerodynamic design tradeoffs.

**Tools:** Python, numerical optimization, aerodynamics, Hess-Smith panel method, NACA airfoils

### [Gravity Battery Winch](https://github.com/nolantopper/Winch-PID-Controller)

Designed and built a scale translating-drum winch for a gravity battery concept with Renewell Energy. The system used PID control to synchronize drum rotation and carriage translation for repeatable cable spooling.

**Tools:** PID control, mechatronics, encoders, motor control, CAD, rapid prototyping, system integration

### [GPS Altitude Mapping](https://github.com/nolantopper/GNSS-Sensor-Fusion)

Implemented an Extended Kalman Filter to fuse GPS altitude and barometric pressure data for smoother elevation estimates, then compared the filtered result against fused location data.

**Tools:** Python / MATLAB, Kalman filtering, sensor fusion, GPS, barometric data
